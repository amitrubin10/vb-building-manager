# 🔄 פרומפט להמשך שיחה ב-Claude Code

**מטרה:** העתק והדבק את הפרומפט הזה בשיחה ראשונה ב-Claude Code על מחשב חדש (או אחרי הפסקה ארוכה) כדי לחזור לאותו מקום שבו עצרנו.

---

## 📌 איפה אנחנו עומדים? (עדכון 2026-04-30)

**הושלם:**
- ✅ FIFO חוצה-מקורות — כל כסף נכנס (בנק/אשראי/ידני) הולך לחוב הישן ביותר אוטומטית
- ✅ תצוגת כרטסת — שלב 1 (קריאה בלבד): כפתור מתג בכרטיס דירה בין `📅 חודשית` ל-`📋 כרטסת`

**הצעד הבא (שלב 2 של הכרטסת):**
- עקיפה ידנית של זיכויים — פופאפ עם אפשרויות פיצול, תג 🔒 על שורות עם override
- אחסון: `vaad_credit_overrides` ב-localStorage
- ראה פירוט מלא בסעיף 11 של `PROJECT-MEMORY.md`

## מה לעשות במחשב החדש (סדר פעולות)

### 1. קלון את ה-repo
```powershell
git clone https://github.com/amitrubin10/vb-building-manager.git
cd vb-building-manager
```

### 2. ודא שיש לך את הגיבוי של הנתונים (JSON)
- הוצא גיבוי מהמחשב הישן: `🏢 הגדרות הבניין → גיבוי ושחזור → ↓ ייצוא`
- העבר את ה-JSON למחשב החדש (אימייל, OneDrive, USB...)

### 3. פתח את הדשבורד וייבא את הנתונים
- לחיצה כפולה על `index.html` בתיקייה
- `🏢 הגדרות הבניין → גיבוי ושחזור → ↑ ייבוא`
- בחר את ה-JSON

### 4. הפעל Claude Code
```powershell
claude
```

### 5. הדבק את הפרומפט למעלה (מ-`>>>` עד `<<<`)

### 6. ודא שהמודל קרא נכון
המודל אמור לאשר לך:
- "קראתי את `PROJECT-MEMORY.md` ואת `מדריך-למשתמש.md`"
- מתאר את הפרויקט בקצרה (FIFO + כרטסת)
- מציין שאנחנו לפני שלב 2 של הכרטסת
- מכיר את 3 ההחלטות UX ואת תרחיש דירה 19

אם משהו חסר/לא נכון — תקן אותו לפני שתמשיך לפיתוח.
